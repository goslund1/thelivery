# FH5 & FH6 Upgrades Hierarchy  
  
Both games share the same underlying pattern of part types, parts, and the general division of how these are accessed and what their cost is. There is some variability to whether a particular car has some of the sections or parts available, but we will work from a maximalist approach and include all options since we can’t be sure which car is being added.

## EV vs ICE slot availability

EV cards replace Engine + Clutch + Transmission with Motor + Battery. Everything else (Platform and Handling, Driveline, Differential, Tires, Aero, Conversions) applies to both.

There is no hard EV/ICE mode on a card — each upgrade slot is independently set to Not Available per car. Some EVs can equip a transmission (e.g. Rimac Concept in FH5 with a 4-speed Drift transmission), so Motor + Battery + Transmission can coexist on the same card.

**FH5:** Motor and Battery upgrades live in the Engine section (inline tier upgrades for the car’s existing electric motor/battery).

**FH6 adds:** Motor and Battery Swap in Body Kits and Conversions — the same Engine Swap / Drivetrain Swap pattern applied to EVs. Lets you drop a different electric powertrain into any compatible car.
