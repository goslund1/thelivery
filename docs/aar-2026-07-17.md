## OG Maker font library

**Problem:** Claude had picked Bebas Neue and Oswald without discussion; font choice was also
conflated with the rendering style (POSTCARD/SIGNAL/GHOST), so there was no way to use a
different font within a given treatment.

**Design settled:** font and rendering style are independent.
- **Style** (POSTCARD / SIGNAL / GHOST) sets the rendering treatment (backdrop, opacity) and
  provides a default font.
- **Font** is an optional per-box override — selecting "— style default —" falls back to the
  style's built-in font.
- Rotate/shear distortions are always per-box, unaffected by either.
