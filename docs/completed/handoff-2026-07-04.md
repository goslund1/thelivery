## Hardcoded color audit — DONE

All action/state colors across components replaced with theme CSS variables:
- `Gallery.vue` — thumb delete confirm buttons (danger/gold)
- `EditBar.vue` — tl-red/yellow/green traffic-light dock buttons, pill save hover (`var(--ink)` not `#111`)
- `ImagePicker.vue` — mgr-delete/add/cancel buttons
- `FactoidPanel.vue` — delete hover, pill × hover

All use `var(--danger)`, `var(--danger-bright)`, `var(--success)`, `var(--success-bright)`, `var(--success-unlit)`, `var(--gold)` with `color-mix()` for tinted fills and glow shadows. Remaining hardcodes in `ThemeBuilder.vue` and `ColorSwatch.vue` are intentional — they're literal hex values being *edited*, not display colors.
